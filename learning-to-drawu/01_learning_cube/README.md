# Learning Cube

The learning cube has three axes:
- X-axis: General drawing progression
- Y-axis: Skill fields
- Z-axis: Learning styles

A learner can move forward, sideways, upward, backward, or diagonally.

The goal is relational skill growth:
line connects to form, form to perspective, perspective to expression, expression to composition.

Cube task formula:
`Drawing task = progression stage + skill field + learning style`
