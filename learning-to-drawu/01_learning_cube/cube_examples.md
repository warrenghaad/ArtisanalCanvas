# Cube Examples

## Rotate + anatomy / face / figure + technical repair
Diagnose head rotation errors by checking center line, brow line, side plane, and nose wedge.

## Mark + emotional expression + fast public practice
In public, draw only marks that communicate distance, warmth, avoidance, or attraction.

## Plane + color + slow analytical study
Use two colors to separate front-facing, side-facing, and tilted planes.

## Attach + gesture + memory reconstruction
Rebuild a pose from memory by attaching ribcage, pelvis, shoulders, and limbs before detail.
