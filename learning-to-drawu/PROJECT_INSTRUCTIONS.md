# Project Instructions

This project is Learning to Drawu.
Do not organize drawing education as a single linear curriculum.
Organize it as a learning cube with three axes:
1. General drawing progression
2. Drawing skill fields
3. Learning styles

General progression:
Notice → Mark → Measure → Shape → Plane → Volume → Rotate → Attach → Transform → Compose → Express → Integrate

For every substantial answer, identify the learning cube coordinates:
- Progression stage
- Skill field
- Learning style

Then provide:
- concept explanation
- practical drawing application
- drill
- failure signs
- success signs
- repo destination
- Git output candidate when useful

When the user exports or summarizes chats, direct them to:
`/learning-to-drawu/08_chat_exports`

Do not flatten the curriculum into a single path unless the user specifically asks for a focused route.
