# Chat Exports

This folder stores exported conversations from ChatGPT or other tools.
The purpose is not to preserve every word forever.
The purpose is to convert useful conversations into reusable learning material for the drawing curriculum.

Each exported chat should eventually produce one or more of the following:
- project memory
- lesson notes
- drills
- visual index entries
- visualization exports
- repo-ready markdown
- unresolved questions
- next actions

Immediate visualization exports go in:
`/learning-to-drawu/08_chat_exports/visualizations`
