# Chat Export Directions

When a chat contains useful drawing material, export it into this folder.

## Step 1: Save the Raw Chat
Place the full exported chat in:
`/learning-to-drawu/08_chat_exports/raw_chats`

Use this filename format:
`YYYY-MM-DD_short-topic-title.md`

Example:
`2026-05-20_learning-cube-learning-styles.md`

## Step 2: Create a Processed Summary
Create a summary in:
`/learning-to-drawu/08_chat_exports/processed_summaries`

Use the same filename stem:
`2026-05-20_learning-cube-learning-styles_summary.md`

## Step 3: Extract Lessons
If the chat contains teachable material, extract it into:
`/learning-to-drawu/08_chat_exports/extracted_lessons`

Each extracted lesson should include:
- concept
- why it matters
- drawing application
- drill
- related skill fields
- related learning cube coordinates

## Step 4: Extract Drills
If the chat contains exercises, place them in:
`/learning-to-drawu/08_chat_exports/extracted_drills`

Each drill should include:
- name
- time limit
- materials
- skill focus
- learning style
- steps
- failure signs
- success signs
- next variation

## Step 5: Create Git Output Candidate
If the chat produced repo-ready text, place it in:
`/learning-to-drawu/08_chat_exports/git_output_candidates`

This is the staging area before moving material into the main curriculum folders.

## Step 6: Export Visualizations Immediately
If the chat or workflow produces visualization files, place them in:
`/learning-to-drawu/08_chat_exports/visualizations`
